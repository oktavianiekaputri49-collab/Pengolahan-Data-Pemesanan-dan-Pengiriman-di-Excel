# Pengolahan-Data-Pemesanan-dan-Pengiriman-di-Excel
Mini project menggunakan Microsoft Excel untuk mengolah data pemesanan dan menentukan lama pengiriman berdasarkan domisili. Data diproses menggunakan referensi sehingga lebih terstruktur dan akurat. Project ini melatih pengolahan data operasional secara efisien.
